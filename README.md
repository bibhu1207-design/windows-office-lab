\# Windows Office IT Lab



\## Overview

This project simulates a basic office IT environment using VirtualBox, Windows 10, and Linux Mint. The objective was to practice virtualization, user management, networking, troubleshooting, and Git/GitHub workflow.



\---



\## Technologies Used

\- Windows 10

\- Windows 11

\- Linux Mint

\- Oracle VM VirtualBox

\- Git

\- GitHub



\---



\## Skills Demonstrated

\- Virtual machine setup

\- User account management

\- Basic networking

\- System troubleshooting

\- Git version control

\- Documentation

\# Troubleshooting



\## Issue

System lag due to limited RAM while running multiple virtual machines.



\## Solution

Reduced VM memory allocation and used only one VM at a time.
